    • Defined in api.ts:196

    Initializes a new instance of the NsApi class.

    Parameters

    • userAgent: string

      A string identifying you to the NationStates API. Using the name of your main nation is recommended.

    • Default value delay: boolean = true

      Whether a delay is introduced before API and telegram requests. Defaults to true.

    • Default value apiDelayMillis: number = 600

      The delay before API requests in milliseconds. Defaults to 600.

    • Default value nonRecruitTgDelayMillis: number = 60000

      The delay before non-recruitment telegram requests in milliseconds. Defaults to 60000.

    • Default value recruitTgDelayMillis: number = 180000

      The delay before recruitment telegram requests in milliseconds. Defaults to 180000.

    • Default value cacheApiRequests: boolean = true

      Whether API requests should be cached. Defaults to true.

    • Default value cacheTime: number = 900

      The number of seconds that API requests should stay cached. Defaults to 900.

    • Default value allowImmediateApiRequests: boolean = true

      Allows API requests immediately after the API is initialized.

    • Default value allowImmediateTgRequests: boolean = true

      Allows telegram requests immediately after the API is initialized.

    Returns NsApi
